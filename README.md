
# 🌸 MAHIRU AI

Mahiru AI is an intelligent virtual companion designed to feel natural, supportive, and emotionally aware.

## Vision
A virtual friend that can:
- 🎤 Talk in Hindi & English
- 😊 Understand mood and emotions
- 🖤 Beautiful black UI
- 👧 Anime-style avatar
- 🧠 Remember conversations
- 🌐 Work online and offline
- 📱 Help with supported phone tasks (with permission)
